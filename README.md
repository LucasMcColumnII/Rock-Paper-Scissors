# **Rock! Paper! Scissors!**

This project was inspired from the YouTube channel called "Programming with Mosh".
It works like the traditional game of rock, paper, scissors but from the programming level using Python.
You will find some additional features that were implemented into this program version of Rock! Paper! Scissors on
the Features Included page.

## **Here's how to execute the program**

**Step 1: Download Visual Studio Code**

```
https://code.visualstudio.com/download
```

**Step 2: Download Python (Preferably Python 3)**

```
https://www.python.org/downloads/
```

**Step 3: Access this repository**

For cloning, opening in GitHub Desktop or downloading ZIP file, here's where to locate those features:

Click on this repository and you can locate these features from the "<> Code" button.

[1] Cloning Repository

```
HTTPS: https://github.com/LucasMcColumnII/Rock-Paper-Scissors.git
SSH: git@github.com:LucasMcColumnII/Rock-Paper-Scissors.git
CLI: gh repo clone LucasMcColumnII/Rock-Paper-Scissors
```

[2] Open in GitHub Desktop

[3] Download ZIP

**Step 4: Install VS Code ~ Python Extensions**

```
Pylance v2025.10.4
Python v2025.18.0
Python Debugger v2025.16.0
```

**Step 5: Execute the program**

```
[Your-OS]:Rock-Paper-Scissors [your-username]$ python3 rps.py
```
