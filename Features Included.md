# **Features Included**

**Player Mode Selection**

There are two modes that can be played: User-to-User and User-to-COM

[1] User-to-User

This feature is available for two local players to make a selection of letters r, p, or s to represent
rock, paper or scissors respectively.

[2] User-to-COM

This feature is available for the user to play against the computer, where the user will make the choice between
the three options and the computer will randomly assign a choice.

**Scoring:**
The game is played 5 times, where for each mode has the scoring set in this fashion

Both the User-to-User and User-to-COM player modes will do the following:

```
- If User 1 (User-to-User) or Player 1 (User-to-COM) wins, they will earn a point (+1), whereas User 2 or 
  COM will lose a point (-1)
- If User 2 (User-to-User) or COM (User-to-COM) wins, they will earn a point (+1), whereas User 1 or 
  Player 1 will lose a point (-1)
- If there is a tie, the scores will not change
```

**Concluding Results**

After the game ends, the program will print out the final score of both players
